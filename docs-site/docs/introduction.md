# Saola

Saola is a privacy-focused, cross-platform desktop API client for macOS, Windows, and Linux. Named after the rare "Asian Unicorn," Saola emphasizes **privacy, speed, and user-owned data**.

## Why Saola

Unlike competitors that force data into their clouds, Saola ensures your API collections and sensitive environment variables stay **in your control**. Sync to your own infrastructure—**Google Drive** or **AWS S3**—with optional **end-to-end encryption (E2EE)** so your data is never exposed in plain text on remote servers.

## Key Features

- **Collection management** — Create collections and folders, organize requests with context menus
- **Request builder** — Full HTTP support (GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS) with params, headers, and body
- **Environments & variables** — Use `{{variable}}` interpolation in URLs, headers, and body
- **Code snippets** — Generate cURL, fetch, and Python snippets with one click
- **Request logs** — Verbose view of outgoing requests and incoming responses
- **Sync & storage** — Local storage, S3, or Google Drive with optional encryption
- **Postman import** — Import Postman v2.1 collections

## Supported Platforms

- **Windows** — `.msi` or `.exe` installer
- **macOS** — `.dmg` (Intel and Apple Silicon)
- **Linux** — `.deb` (Debian/Ubuntu) or `.rpm` (Fedora/RHEL)
