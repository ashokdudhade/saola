# Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+Enter` (Mac) / `Ctrl+Enter` (Win/Linux) | Send request |
| `Cmd+S` (Mac) / `Ctrl+S` (Win/Linux) | Save request to collection |
| `Cmd+\` (Mac) / `Ctrl+\` (Win/Linux) | Toggle sidebar |
| `Cmd+P` (Mac) / `Ctrl+P` (Win/Linux) | Open global search |
