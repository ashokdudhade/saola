# Postman Import

Saola can import collections from Postman. Open **Settings** → **Storage** and use the **Postman import** option.

## Supported Format

- **Postman Collection v2.1** — Export your collection as JSON (Collection v2.1) from Postman.

## Import Options

- **Append** — Add imported collections to your existing collections.
- **Replace** — Replace your current collections with the imported data.

Choose the Postman v2.1 JSON file when prompted. Saola merges the imported structure into local storage.
